# Ambrosai Wellness Website (Starter)

Multi-page responsive static website for Ambrosai Wellness.

## Pages
- Home (`index.html`)
- About Business (`about-business.html`)
- About Us (`about-us.html`)
- Programs (`programs.html`)
- Recipes (`recipes.html`)
- Reels (`reels.html`)
- Contact (`contact.html`)

## Run locally
Open `index.html` directly or use a static file server:

```bash
python3 -m http.server 8080
```

## Notes
- Color theme is inspired by the provided logo (green/blue + light background).
- Instagram Reels API integration is not active in this static version because it requires Meta app credentials and a backend token flow.
- Current contact flow directs users to Instagram DMs and WhatsApp, as requested.
